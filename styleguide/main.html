Button

Button / Anchor element.

Default button size is l. Default button type is secondary.

Button properties:

  • button-type: "primary" | "secondary"

  • button-size: "s" | "l"

  • button-class: string

  • attributes: hash | string

  • tag: "button" | "a"

  • label: string
